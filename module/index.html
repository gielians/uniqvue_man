Modules

Module treten an verschiedenen Stellen im Manager auf. In der Session sind alle vorhanden um eine Agenda zu füllen, Die Toolboxmodule sind entsprechend in der Toolboxkonfiguration zu finden und wir gehen im Kapitel Toolbox noch einmal darauf ein.

Hier ist eine Übersicht der Module aus der Session und was sie können.

Singleview Module:

Diese Module können je nasch Display Setup an eine Position gezogen werden und erscheinen später im Showroom auf dem enstsprechenden Display. Alle Singleview Module haben konsistente Properties die hier aufgelistet werden.

Background Image: Über den Asset Browser wird hier ein Hintergrundbild für dieses Modul geladen.

Size: Hier wird die Größe der Darstellung des Fensters angegeben. Der Default Wert ist 0.75. Der Wert 1 entspricht Fullsize.

Title: Vergeben sie einen Titel für dieses Modul

Showtitle: Setzen sie einen Haken wenn dieser Titel auch später angezeigt werden soll.

Folgenden Module sind alles Singleviewmodule. Bei der Auflistung folgen dann noch die individuellen Properties des jeweiligen Moduls um sie zu vervollständigen.

Browser Modul:

• Show Header - Der gesetzte Haken zeigt die URL Leiste, Bookmarks etc. An. • URL - Hier kann man die Start URL eintragen mit der das Browsermodul starten soll, z.B. https://www.google.de

• Bookmarks 01 - 05 - Hier stehen 5 Bookmarks zur Verfügung in denen ich auch URLs vergeben kann, diese werden als kleine Icons generiert und oben rechts im Browserfenster angezeit und dienen zur direkt Wahl.

HTML Modul:

• HTML File - laden sie über den Asset Browser ein HTML File zur Ansicht.

• Transparancy - der gesetzte Haken schaltet den Hintergrund Transparent.

Bulletlist Modul:

• Title - in einem Editor vergeben sie hier einen Titel für die Bulletlist

• Bullet - erzeugen sie hier einen neuen Punkt in der Bulletlist

Headline Modul:

• Headline - erstellen sie eine Headline

Pause Modul:

• Titel - in einem Editor vergeben sie hier einen Title

• Bullet - erzeugen sie hier einen neuen Punkt der in der Pausenliste

• Timer - geben sie hier die länge der Pause an. Diese läuft dann wie es ein Timer tun soll ab bis die Pause um ist.

Live Input:

• Input Source - wählen sie aus einem Dropdownmenu eine aus den verfügbaren Quellen aus

Video Modul:

• Video File - Öffnet den Asset Browser um ein Video auszuwählen.

• Autostart - ist der Haken gesetzt startet das Video automatisch

• Repeat Video - ist der Haken gesetzt läuft das Video in einem Loop

Share Price:

• Share Price Symbol - geben sie hier die WKN des Unternehmens an um den entsprechenden Share Price angezeigt zu bekommen.

Slide Show:

• Folder Path - öffnet den Asset Browser in dem sie dann den Pfad zur Slideshow auswählen können.

• Show Navigation - der gesetzte Haken zeigt die Navigation an

• Navigation inside Frame - gesetzteer Haken zeigt die Navigation im Bild an.

Multiview Module:

Je nach Set up des Showrooms verhalten sich die Multiview Module anders. Das wirkt sich hauptsächlich auf die Anordnung der Fenster aus und wir angezeigt sobald ich das Modul in die Session per Drag n Drop platziere. Zu dem wird in den Properties angezeigt was in ihrem Set up geht. Grundsätzlich gilt aber das Multiview Module mehr als ein Fenster haben und dadurch zusätzliche Funktionalität mit sich bringen.

Storyboard Modul:

Kann aus bis zu 3 Fenstern bestehen. Das Center Fenster zeigt Slides an die ich mit einer Wischgeste nach rechts und links bewegen kann. Ein Slide welches ich so bewegt habe sammelt sich entsprechend rechts oder links in einem Fenster an, und ordnet sich in ein Grid ein.

• Slide Folder - öffnet den Asset Browser um einen Pfad zum Ordner mit den Slides auszuwählen

• Storyboard Left - Je nach Setup und Platzierung ist der Haken gesetzt. Außerdem kann ich für dieses Fenster separat ein Background Image mit dem Assetbrowser auswählen.

• Storyboard Right - Je nach Setup und Platzierung ist der Haken gesetzt. Außerdem kann ich für dieses Fenster separat ein Background Image mit dem Assetbrowser auswählen.

Dashboard Modul:

• Input Source 01 - 06 - Hier lassen sich verschiedene Quellen auswählen und mit einem Dashboard Layout darstellen.

Challanges Modul:

Ein Spezial Modul welches Kundenspezifisch erklärt wird.
