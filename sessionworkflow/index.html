Session Workflow

Mit den konsistenten Bedienelemente die sich rechts aussen am Dokumentfenster befinden kann man einfach eine Session bearbeiten, erstellen, duplizieren oder löschen. Zur Erinnerung noch einmal die Erklärung der Icons von oben nach unten.

Der Pfeil der nach links zeigt führt zurück in das Menu von welchem man gestartet ist. Der Stift ist das Symbol um eine ausgewählte Session zu bearbeiten. Das Dokument mit dem Pluszeichen ist zum erstellen einer neuen Session. Der Dokumentenstapel darunter ist zum duplizieren einer ausgewählten Session. Und zu guter letzt löscht man mit dem Mülleimer Symbol eine ausgewählte Session.

Wir erstellen jetzt eine neue Session in dem wir auf das Dokument mit dem Pluszeichen klicken. Mit einem doppleklick auf den Namen können wir diesen editieren und vergeben einen Namen wie zum Beispiel „meine Session“. Diese Session ist erst mal leer und hat außer einer Default toolbox und einem default style keine Inhalte. Das ändern wir in dem wir entweder auf den Stift zum bearbeiten klicken, oder ganz recht im Session Inspector auf Edit Session. Beides öffnet einen neuen Tab und wir befinden uns innerhalb der Session. Der Name unserer Session steht dazu im Tab, da wir beliebig viele Tabs mit Sessions öffnen können.

Alle Bereiche sind beschriftet und folgen nun mit einer Positionsangabe.

Das Fenster ist viergeteilt. Links oben befindet sich der Agenda Tree, darunter die Agenda Module Repository. In der Mitte der Agenda Item Editor und rechts aussen der Property Editor.

Um eine Agenda anzulegen starte ich mit dem erstellen einzelner Punkte im Agenda Tree in dem ich einfach auf den Button Add Agenda Item anklicke. Darauf erschient in der ein neuer Punkt der Empty heißt den ich wie einen Namen mit einem Doppelklick ändern kann. Zu dem Punkt gehört im Agenda Item Editor eine neue Ansicht des Display Setups.

Um diesen Punkt jetzt mit Inhalt zu befüllen sind zwei Schritte nötig.

  1. Ich ziehe aus dem Agenda Module Repository ein Modul in platziere es per drag and drop auf das Display Setup an die Position wo ich es gerne haben möchte. Eine grüne Umrandung zeigt mir dabei die Möglichkeiten an. Es gibt, je nach Setup verschiedene Module und Posiotionsmöglichkeiten. Ihr Setup können sie hier einsehen und welches Modul was kann sehen die hier. (Mit diesem Link kommen sie in den Agendabereich des Showrooms in Teil 3 des Manuals und können sehen was aus dieser Aktion wird.)
  2. Wiederholen sie diesen Schritt bis sie ihre Agenda vollständig haben.
  3. Befüllen sie jetzt ihre Module mit Inhalten. Dazu klicken sie auf das von Ihnen positionierte Modul und im rechten Fenster Property Editor sehen sie je nach Modul welche Properties dieses Modul anbietet. Im Kapitel Modul können sie nachschlagen was jedes Modul kann und wie sie die Inhalte rein laden.
  4. Wenn sie die Agenda stehen haben und alle Inhalte gefüllt sind, dann speichern sie die Session und gehen zurück in die Session Liste

In der Sessionliste können sie noch eine Toolbox auswählen und einen Style auswählen um der Session den letzten Schliff zu verpassen. Was die Toolbox ist und was sie dort alles machen können erfahren sie hier. Dasselbe zum Thema Style erfahren sie hier. Wie sie sich selber eine Tollbox erstellen und konfigurieren erfahren sie im Kapitel Toolbox Workflow.

Im Kapitel Style Workflow erfahren sie wie sie eine Style erstellen und konfigurieren.

Vergeben sie noch die Category in der Sessionliste unter der sie die Session im Showroom wiederfinden.

Setzen sie sum schluss einen Haken bei released und sie finden im Showroom Dashboard (LINK) unter der geählten Category ihre Session die sie dort nun laden können. Wie das im Showroom geht erfahren sie hier.
